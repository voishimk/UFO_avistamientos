Google Cloud como BigQuery, DataPrep y Looker Studio.

Comenzar

Ingesta de datos

Durante el proceso de ingesta de datos, se comenzó transformando el archivo original en formato Excel a un archivo CSV, con el objetivo de asegurar la compatibilidad y facilitar su procesamiento en las herramientas de Google Cloud. A continuación, se creó un bucket en Cloud Storage, donde se almacenó el archivo CSV. Posteriormente, se accedió a BigQuery para crear un dataset denominado "ovni", el cual serviría como contenedor lógico para organizar y estructurar los datos. Este flujo permitió establecer una base sólida para la posterior limpieza, análisis y visualización de los datos.

Consultas SQL en BigQuery

Con la información ya depurada, se procedió a realizar consultas SQL dentro de BigQuery, extrayendo los principales patrones y tendencias relevantes.


#Primera consulta SQL - TOP 5 Avistamientos según forma
SELECT UFO_shape,
COUNT(*) AS Sightings
FROM qwiklabs-gcp-04-85b0703cd55b.ovni.avistamientos 
WHERE country = 'us'
GROUP BY UFO_shape
ORDER BY Sightings DESC
LIMIT 5;

1️⃣ Identificar los 5 tipos de avistamientos con más cantidad de apariciones


#Segunda consulta SQL - Cantidad de avistamientos por año
SELECT EXTRACT(YEAR FROM DATETIME(Date_time)) AS Year, COUNT(*) AS Sightings
FROM qwiklabs-gcp-04-85b0703cd55b.ovni.avistamientos
WHERE country = 'us'
GROUP BY Year
ORDER BY Sightings DESC;

2️⃣ Cantidad de avistamientos por año


#Tercera consulta SQL - Cantidad de avistamientos según estado en EE.UU
SELECT `stateProvince` AS State, COUNT(*) AS Sightings
FROM qwiklabs-gcp-04-85b0703cd55b.ovni.avistamientos
WHERE country = 'us'
GROUP BY State
ORDER BY Sightings DESC;
